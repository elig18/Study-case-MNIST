# Study-case-MNIST

##  Projet
Étude de cas : Classification d'images de chiffres manuscrits (dataset MNIST) en 10 catégories (0 à 9) à l'aide de réseaux de neurones convolutifs (CNN).

## Objectif
Concevoir et entraîner un modèle de Deep Learning capable de reconnaître et classer des chiffres manuscrits avec une haute précision.

## Approche
- Prétraitement des données (normalisation, reshape)
- Architecture de réseau de neurones convolutifs (CNN)
- Entraînement et évaluation du modèle
- Optimisation des hyperparamètres

## Dataset
**MNIST** (Modified National Institute of Standards and Technology)
- 60 000 images d'entraînement
- 10 000 images de test
- Images en niveaux de gris de 28x28 pixels
- 10 classes (chiffres 0 à 9)

## Technologies utilisées
- Python
- PyTorch / TensorFlow / Keras
- NumPy / Pandas
- Matplotlib pour les visualisations
